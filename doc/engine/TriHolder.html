TriHolder

engine.TriHolder
class TriHolder(val pos: GridPos)

The class TriHolder represents the static slot (holder) that holds a triangle tile in a hexagon, as an abstract concept.

A TriHolder object is potentially mutable: its state can change as triangle tiles enter and exit it.
